## KEN'S PHOTO GALLERY

## Author
Kennedy Karuri

## Description

Ken's Photo-Gallery is an admin managed page that allows the user to visit and view cool and amazing photos posted by the admin

## Features
* View different photos that interest you.
* Click on a single photo to expand it and also view the details of the photo. 
* Search for different categories of photos. (ie. Cars, Nature)
* Copy a link to the photo to share with your friends.
* View photos based on the location they were taken.

## Technologies used
* Python3.9
* Django
* Html
* Bootstrap
